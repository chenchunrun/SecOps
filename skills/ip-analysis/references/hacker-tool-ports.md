# 黑客工具默认端口参考

常见黑客工具、C2 框架、RAT 和恶意软件使用的默认端口。用于端口扫描时识别潜在威胁。

> **重要提醒**：端口开放 ≠ 确认恶意。很多合法服务也使用这些端口。
> - 31337：可能是 Elite、游戏服务器、或任何自定义服务
> - 4444：可能是 Kerberos、KDC 或其他合法服务
> - 8080：最常见的 HTTP 代理端口
>
> **必须结合服务指纹验证**（JARM、SSL 证书、HTTP 响应、Banner）才能确认是恶意 C2。

## C2 服务指纹验证

> **关键**：端口开放只是线索，以下指纹才是确认证据。

### JARM 指纹

JARM 是 TLS 服务器指纹工具，通过发送特定 TLS Client Hello 获取服务器响应特征。

```bash
# 安装
pip install jarm

# 使用
python -m jarm <IP>:<PORT>
```

| C2 框架 | JARM 指纹 | 可信度 | 说明 |
|--------|----------|-------|------|
| Cobalt Strike (Java 11+) | `07d14d16d21d21d00042d41d00041de5fb3038104f457d92ba02e9311512c2` | 中 | 也匹配其他 Java 应用 |
| Cobalt Strike (旧版) | `07d14d16d21d21d07c42d41d00041d24a458a375eef0c576d23a7bab9a9fb1` | 中 | Java 应用通用 |
| Sliver HTTP/S | `3fd21b20d00000021c43d21b21b43d41226dd5dfc615dd4a96265559485910` | 中高 | 较为特定 |
| Havoc | `3fd21b20d00000021c43d21b21b43de0a012c76cf078b8d06f4620c2286f5e` | 中高 | 较为特定 |

> **警告**：Cobalt Strike JARM 实际上是 Java TLS 实现的通用指纹，会匹配很多合法 Java 服务。单独使用有较高误报率。

### SSL 证书特征

```bash
# 检查证书
echo | openssl s_client -connect <IP>:<PORT> 2>/dev/null | openssl x509 -noout -serial -issuer -subject -dates
```

| C2 框架 | 证书特征 | 可信度 | 说明 |
|--------|---------|-------|------|
| Cobalt Strike | Serial: `146473198` (0x8bb00ee) | 高 | 默认证书，未更改配置的脚本小子 |
| Cobalt Strike | Subject: `CN=localhost, O=cobaltstrike` | 高 | 默认配置 |
| Cobalt Strike | Subject: `CN=gmail.com` (自签名) | 中 | Malleable C2 伪装 |
| Sliver | 随机生成的自签名证书 | 低 | 无特定特征 |
| Metasploit | Subject 包含 `meterpreter` | 高 | 未更改配置 |

### HTTP 响应特征

```bash
# 检查响应头
curl -sI http://<IP>:<PORT>/ 2>/dev/null | head -20
curl -sI https://<IP>:<PORT>/ -k 2>/dev/null | head -20
```

| C2 框架 | 响应特征 | 可信度 | 说明 |
|--------|---------|-------|------|
| Sliver | `404 Not Found` + `Cache-Control: no-store, no-cache, must-revalidate` | 高 | 默认配置 |
| Sliver | Server 显示 Apache/PHP 但行为异常 | 中 | 伪装特征 |
| Cobalt Strike | 特定 URI 返回 Beacon 配置 | 高 | Malleable C2 配置 |
| Mythic | 7443 端口返回 React 应用 | 高 | Web UI |

# 风险指标说明

## 端口风险分类

### 高风险端口

暴露这些端口意味着直接的安全风险，应立即处理。

| 端口 | 服务 | 风险说明 |
|------|------|----------|
| 23 | Telnet | 明文传输，易被嗅探 |
| 445 | SMB | 永恒之蓝等漏洞利用 |
| 3306 | MySQL | 数据库暴力破解/注入 |
| 5432 | PostgreSQL | 数据库攻击 |
| 27017 | MongoDB | 无认证数据泄露 |
| 6379 | Redis | 未授权访问/RCE |
| 9200 | Elasticsearch | 数据泄露/RCE |
| 11211 | Memcached | DDoS 放大/数据泄露 |
| 502 | Modbus | 工控协议，无认证 |
| 102 | S7comm | 西门子工控协议 |

### 中风险端口

需要评估是否有业务必要性，建议限制访问。

| 端口 | 服务 | 风险说明 |
|------|------|----------|
| 22 | SSH | 暴力破解目标 |
| 3389 | RDP | 暴力破解/漏洞利用 |
| 5900 | VNC | 远程控制 |
| 21 | FTP | 匿名访问/暴力破解 |
| 25 | SMTP | 垃圾邮件中继 |
| 1433 | MSSQL | 数据库攻击 |
| 1521 | Oracle | 数据库攻击 |
| 8080 | HTTP-Alt | 管理界面暴露 |
| 8443 | HTTPS-Alt | 管理界面暴露 |

### 低风险端口

常规业务端口，但仍需关注配置安全。

| 端口 | 服务 | 说明 |
|------|------|------|
| 80 | HTTP | 标准 Web 服务 |
| 443 | HTTPS | 加密 Web 服务 |
| 53 | DNS | 域名解析 |
| 110 | POP3 | 邮件服务 |
| 143 | IMAP | 邮件服务 |

## Banner 风险特征

### 敏感信息泄露

| Banner 特征 | 风险说明 | 等级 |
|-------------|----------|------|
| `password:` | 可能泄露凭据 | 高 |
| `root@` | 暴露用户名 | 中 |
| `debug=true` | 调试模式开启 | 中 |
| `phpinfo()` | PHP 配置泄露 | 中 |
| `error:` + 路径 | 路径泄露 | 低 |

### 默认配置特征

| Banner 特征 | 说明 | 等级 |
|-------------|------|------|
| `Welcome to nginx!` | 默认页面 | 低 |
| `Apache2 Ubuntu Default Page` | 默认页面 | 低 |
| `It works!` | Apache 默认 | 低 |
| `401 Unauthorized` + `WWW-Authenticate: Basic` | 基础认证 | 中 |

### 无认证服务特征

| Banner 特征 | 说明 | 等级 |
|-------------|------|------|
| MongoDB: 无认证提示 | 无密码 | 高 |
| Redis: `-NOAUTH` | 无密码 | 高 |
| Elasticsearch: 直接返回 JSON | 无认证 | 高 |

## 风险场景示例

### 场景 1: 暴露的 Redis

```
发现: 1.2.3.4:6379
组件: Redis 5.0.0
Banner: -NOAUTH Authentication required.

风险评分:
- Redis 基础分: +10
- 公网暴露 6379: +20
- 无认证提示（但需密码）: +5
- 总分: 35 → 中风险 🟡
```

### 场景 2: 无认证 MongoDB

```
发现: 5.6.7.8:27017
组件: MongoDB 4.0.0
响应: 直接返回数据库列表

风险评分:
- MongoDB 基础分: +10
- 无认证: +40
- 公网暴露 27017: +20
- 总分: 70 → 严重 🔴
```

### 场景 3: 老旧 Apache

```
发现: 9.10.11.12:80
组件: Apache 2.2.15
Banner: Apache/2.2.15 (CentOS)

风险评分:
- Apache 基础分: +5
- 老旧版本 (2.2.x): +15
- 标准 HTTP 端口: +0
- 总分: 20 → 低风险 🟢
```

### 场景 4: 工控设备

```
发现: 13.14.15.16:502
协议: Modbus
设备: PLC 控制器

风险评分:
- 工控设备基础分: +25
- 公网暴露工控协议: +35
- Modbus 无认证: +40
- 总分: 100 → 严重 🔴
```

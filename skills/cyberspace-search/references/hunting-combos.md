# 网络空间搜索分析连招

## 连招 1: 单IP → 全面展开

```
已知恶意IP: 1.2.3.4

1. ip="1.2.3.4"                    → 获取绑定域名、证书、端口
2. 提取证书CN → ssl="证书特征"      → 证书关联其他IP
3. 提取域名 → domain="xxx.com"     → 域名历史IP、同域资产
4. 威胁情报 → risk_insight         → 获取家族标签
5. 同IP域名统计                     → 判断防弹托管（>100域名=可疑）
```

## 连招 2: 域名 → 基础设施追踪

```
已知恶意域名: evil.duckdns.org

1. domain="evil.duckdns.org"       → 当前解析IP
2. ip="解析IP"                     → 同IP其他域名
3. domain="duckdns.org"            → 同动态DNS服务的其他域名
4. 威胁情报交叉验证                  → risk_insight
```

## 连招 3: 域名入手 → 穿透 CDN

```
已知: target.com (使用CDN)

1. hostname="*.target.com"         → 收集子域名
2. 对每个子域名测试是否有 CDN
3. ssl="target.com"                → 证书关联找源站
4. iconhash 搜索                    → 找同图标的非 CDN 站点
5. 历史 DNS 记录对比
6. 绑定 hosts 验证真实 IP
```

## 连招 4: 组件漏洞 → 批量收割

```
发现目标使用 ThinkPHP 5.0

1. app:thinkphp +ver:5.0 +country:CN     → 同版本资产
2. "X-Powered-By: ThinkPHP" +port:80     → Banner 确认
3. 导出 IP 列表
4. ops_portscan 批量验证存活
5. 漏洞验证
```

## 连招 5: C2 基础设施追踪

```
发现可疑 C2: 5.6.7.8

1. ip="5.6.7.8"                    → 当前服务信息
2. history:5.6.7.8                 → 历史变化（需API）
3. cidr="5.6.7.0/24"               → 同网段是否有更多 C2
4. ssl 证书特征全网搜索
5. JARM 指纹匹配
6. risk_insight 查威胁情报
```

## 连招 6: 基础设施变更检测

```
发现域名IP从 A 变成 B

1. ip="A" 和 ip="B"                → 对比两IP的域名列表
2. 检查是否使用相同证书              → 确认关联
3. 记录变更时间线                    → 追踪基础设施演化
4. 新IP立即加入封禁列表
```

## C 段用途分析模板

```markdown
## C段用途分析: x.x.x.0/24

### 网段性质判断
- [ ] 企业自有网段 (特征: 统一域名后缀、企业设备)
- [ ] IDC/云托管 (特征: 多租户、杂乱域名)
- [ ] 专用托管 (特征: 同类业务，如游戏私服)
- [ ] 恶意基础设施 (特征: 大量403、无域名绑定)

### 高价值目标
| IP | Title | 价值 |
|----|-------|------|

### 攻击面评估
| 类型 | 数量 | 风险 |
|------|------|------|
| VPN入口 | x | 高 |
| 管理后台 | x | 高 |
| 默认页面 | x | 中 |
```

## C段 Title 用途推断

| Title关键词 | 推断用途 | 风险提示 |
|------------|---------|---------|
| BIG-IP, F5, Citrix | 企业负载均衡 | 高价值目标 |
| GlobalProtect, VPN | VPN入口 | 重点攻击面 |
| 登录, Login, Admin | 业务系统入口 | 可尝试弱口令 |
| IIS, Apache默认页 | 未配置服务器 | 配置问题 |
| XAMPP, phpMyAdmin | 开发环境 | 高风险暴露 |
| 医疗, 政务, OA | 行业业务系统 | 敏感数据 |
| 403/404/401 | 受限服务 | 需进一步探测 |
| Index of / | 目录列表 | 信息泄露 |

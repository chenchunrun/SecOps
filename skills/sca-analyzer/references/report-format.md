## 3. 漏洞详情

### 3.1 严重漏洞

**CVE-2021-44906 - minimist 原型污染**

| 属性 | 值 |
|------|-----|
| 组件 | minimist |
| 受影响版本 | < 1.2.6 |
| 当前版本 | 1.2.5 |
| CVSS | 9.8 |
| 类型 | 原型污染 |
| 修复版本 | 1.2.6 |
| 依赖路径 | mkdirp@0.5.5 -> minimist@1.2.5 |

**漏洞描述**:
Minimist 1.2.6 之前版本存在原型污染漏洞，攻击者可通过构造的参数修改 Object.prototype，可能导致任意代码执行。

**修复建议**:
```bash
npm update mkdirp
# 或
npm install minimist@1.2.6 --save
```

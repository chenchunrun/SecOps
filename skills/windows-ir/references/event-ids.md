## 快速查询命令

```powershell
# 查询登录失败
Get-WinEvent -FilterHashtable @{LogName='Security';ID=4625} -MaxEvents 50

# 查询新用户创建
Get-WinEvent -FilterHashtable @{LogName='Security';ID=4720} -MaxEvents 20

# 查询服务安装
Get-WinEvent -FilterHashtable @{LogName='System';ID=7045} -MaxEvents 20

# 查询日志清除
Get-WinEvent -FilterHashtable @{LogName='Security';ID=1102}

# 查询 PowerShell 脚本执行
Get-WinEvent -FilterHashtable @{LogName='Microsoft-Windows-PowerShell/Operational';ID=4104} -MaxEvents 50

# 时间范围查询
$start = (Get-Date).AddDays(-1)
Get-WinEvent -FilterHashtable @{LogName='Security';ID=4624;StartTime=$start}
```

## UTG-Q-1000 (银狐/游蛇)

### 基本信息

| 字段     | 值                                                              |
| -------- | --------------------------------------------------------------- |
| 组织 ID  | UTG-Q-1000                                                      |
| 名称     | 银狐 / 游蛇                                                     |
| 别名     | 银狐、游蛇、谷堕大盗、SwimSnake、SilverFox、树狼                |
| 活跃时间 | 2022 年至今                                                     |
| 主要目标 | 财务人员、企业管理人员、电商客服、设计人员                      |
| 归属     | 国内黑产联盟（非单一团伙）                                      |

### 组织描述

UTG-Q-1000 是国内最活跃的黑产组织之一，实际上是多个黑产团伙共用的木马家族和攻击框架。该组织以"银狐"木马为核心，通过仿冒软件下载站、钓鱼邮件、即时通讯软件传播远控木马，主要目标是企业财务人员和管理人员。

### 域名特征

```regex
# 仿冒软件下载站
(?i)(wps|office|finalshell|xshell|navicat|chrome|firefox)[-_]?(cn|zh|download|setup)?
(?i)(todesk|teamviewer|anydesk|sunlogin)[-_]?(cn|download)?
(?i)(flash|adobe|pdf)[-_]?(player|reader|cn)?

# 随机字符短域名
^[a-z]{4,8}\d{2,6}$

# 含数字后缀
[a-z]+\d{4,}$
```

**偏好 TLD**: `.cn`, `.xyz`, `.top`, `.cc`, `.com`

### URL 参数特征

```regex
member=[a-z0-9]+          # 会员追踪参数
(from|source|channel)=    # 渠道追踪
```

### 钓鱼主题

- 税务稽查通知
- 电子发票
- 补贴公告（工资补贴、育儿补贴）
- 人事调动通知
- 社保/医保认证

### 技术特征

**基础设施**:

- 阿里云 OSS (`*.oss-cn-*.aliyuncs.com`)
- 腾讯云 COS (`*.cos.ap-*.myqcloud.com`)
- 有道云笔记

**攻击技术**:

- 白加黑（DLL 侧加载）
- SEO 投毒
- 多阶段载荷加载
- 合法签名程序滥用

## APT-Q-27 (金眼狗)

### 基本信息

| 字段     | 值                                                   |
| -------- | ---------------------------------------------------- |
| 组织 ID  | APT-Q-27                                             |
| 名称     | 金眼狗                                               |
| 别名     | GoldenEyeDog, Dragon Breath                          |
| 活跃时间 | 2015 年至今                                          |
| 主要目标 | 博彩行业、海外华人、Telegram 用户、加密货币用户      |
| 归属     | 独立黑产团伙，隶属 Miuuti Group                      |

### 组织描述

金眼狗是一个针对东南亚博彩行业、海外华人群体的黑客团伙，业务涵盖远控、挖矿、DDoS 和流量相关。该组织技术水平较高，使用多种语言（.NET、C++、Golang、Delphi）开发恶意软件，免杀能力强。

### 域名特征

```regex
# Telegram 仿冒
(?i)telegram[-_]?(zh|cn|chinese|中文)?
(?i)(potato|signal|whatsapp)[-_]?(cn|download)?
(?i)tele[-_]?gram

# 博彩相关
(?i)(bet|casino|lottery|game)[-_]?\d*
```

**偏好 TLD**: `.com`, `.xyz`, `.cc`, `.net`

### 钓鱼主题

- Telegram 中文版
- 聊天软件下载
- 语言包安装
- 博彩平台

### 技术特征

**C2 端口**: `1445`, `1446`, `6688`, `5780`

**攻击技术**:

- 白加黑（MSI 安装包）
- 注册表存储 Payload
- 钱包窃取（MetaMask 扩展）
- 水坑攻击

## 缅北魔方G

### 基本信息

| 字段     | 值                                      |
| -------- | --------------------------------------- |
| 组织 ID  | MOFANG-G                                |
| 名称     | 缅北魔方 G                              |
| 别名     | 魔方 G, Mofang-G                        |
| 活跃时间 | 2021 年至今                             |
| 主要目标 | 企业财务人员、普通员工                  |
| 归属     | 缅北黑灰产团伙                          |

### 组织描述

缅北魔方 G 是位于缅北的黑灰产团伙，主要通过钓鱼邮件和短信针对企业员工进行金融诈骗。该组织开发了冒充工资补贴、ETC、社保、医保等主题的钓鱼网站。

### 域名特征

```regex
# 中转域名模式
site\d+\.g[a-z]+\.r[a-z]+

# 补贴/社保主题
(?i)(butie|subsidy|shebao|yibao|etc)[-_]?
(?i)(gongzi|salary|buzhu)[-_]?
```

**偏好 TLD**: `.xyz`, `.uho`, `.com`, `.cn`

### 钓鱼主题

- 工资补贴通知
- ETC 欠费提醒
- 社保认证
- 医保服务

### 技术特征

**域名策略**:

- 使用 CNAME 解析至 `site*.g*.r*` 中转服务器
- 域名快速轮换，传播前才上线
- 子域名轮换规避检测

**传播方式**:

- 钓鱼邮件
- 二维码传播
- UA 检测（限制手机端访问）
- 多级跳转、框架嵌套

## Smishing Triad

### 基本信息

| 字段     | 值                                           |
| -------- | -------------------------------------------- |
| 组织 ID  | SMISHING-TRIAD                               |
| 名称     | 短信钓鱼三合会                               |
| 别名     | Smishing Triad                               |
| 活跃时间 | 2023 年至今                                  |
| 主要目标 | 普通用户、银行客户、快递用户                 |
| 归属     | PhaaS 黑产联盟                               |

### 组织描述

专注于短信钓鱼(Smishing)的黑产组织，使用钓鱼即服务(PhaaS)模式运营，主要托管在国内云服务商。该组织通过伪装快递、银行等主题进行大规模短信钓鱼。

### 域名特征

```regex
# 短域名、随机字符
^[a-z]{3,5}\d{1,3}$

# 仿冒快递
(?i)(sf|ems|yd|zt|yto)[-_]?(express|kuaidi)?

# 仿冒银行
(?i)(icbc|ccb|abc|boc|cmb)[-_]?(bank|cn)?
```

**偏好 TLD**: `.cc`, `.xyz`, `.top`, `.icu`

### 钓鱼主题

- 快递签收通知
- 包裹异常
- 银行卡异常
- 账户冻结

### 技术特征

**基础设施**:

- 腾讯云 (AS132203)
- 阿里云 (AS45102)
- 约 80% 钓鱼站点托管于上述两家云服务商

**域名策略**:

- 使用隐私保护
- 域名存活时间短（< 6 小时）
- 一次性域名

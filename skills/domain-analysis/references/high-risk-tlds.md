# 高风险 TLD 列表

## 免费/低价域名（高风险）

这些 TLD 常被滥用，因为注册成本低或免费：

| TLD | 说明 | 滥用原因 |
|-----|------|---------|
| .tk | Tokelau 免费域名 | 无需验证，常见于钓鱼 |
| .ml | Mali 免费域名 | 无需验证 |
| .ga | Gabon 免费域名 | 无需验证 |
| .cf | Central African Republic | 无需验证 |
| .gq | Equatorial Guinea | 无需验证 |

## 国家域名（需关注）

某些国家域名因管理宽松而被滥用：

| TLD | 国家 | 风险说明 |
|-----|------|---------|
| .ru | 俄罗斯 | 恶意软件托管 |
| .su | 苏联（遗留） | 网络犯罪 |
| .cn | 中国 | 需检查 ICP 备案 |
| .cc | Cocos Islands | 常被滥用 |
| .ws | Samoa | 短链服务 |
| .to | Tonga | 短链服务 |

## 合法但需谨慎

这些 TLD 本身合法，但要注意仿冒：

| TLD | 合法用途 | 风险场景 |
|-----|---------|---------|
| .com | 商业网站 | 仿冒知名品牌 |
| .org | 非营利组织 | 仿冒官方机构 |
| .gov | 政府网站 | 几乎不可能被滥用 |
| .edu | 教育机构 | 极少被滥用 |

## 使用建议

1. **TLD 本身不决定威胁**：需结合其他指标综合判断
2. **新注册 + 高风险 TLD = 高度可疑**
3. **关注域名主体**：即使是 .com 也可能是钓鱼
4. **检查 WHOIS 信息**：注册时间、注册人等

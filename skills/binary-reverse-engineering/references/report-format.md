## 第5章：恶意软件用途分析

基于检测到的行为能力，推断恶意软件的主要用途。

### 用途分类表

| 用途类型 | 判定依据（满足任意 2 项） |
|---------|-------------------------|
| **Loader/Downloader** | HTTP 下载 + 进程创建 + 文件写入 |
| **RAT (远控)** | 网络通信 + 命令执行 + 屏幕截图/键盘记录 |
| **Stealer (窃密)** | 浏览器路径 + 钱包路径 + 凭据 API |
| **Ransomware** | 文件遍历 + 加密 API + 扩展名修改 |
| **Miner** | 矿池地址 + 高 CPU 特征字符串 |
| **Botnet** | C2 通信 + 任务下发 + 持久化 |
| **Backdoor** | 反向连接 + 命令执行 + 隐蔽通信 |

### 用途分析模板

```markdown
### 恶意软件用途分析

**主要用途**: [Loader/RAT/Stealer/...]

**功能目的**:
| 功能 | 证据 | 置信度 |
|------|------|--------|
| 下载执行其他载荷 | HttpSendRequest + CreateProcess | 高 |
| 远程桌面访问 | termservice 启用 + vnc.exe | 高 |
| 系统信息收集 | GetComputerNameEx + GetVersionEx | 中 |

**攻击者意图**:
该样本为 Amadey 家族的 Loader，主要用于：
1. 收集受害主机信息并回传 C2
2. 根据 C2 指令下载并执行后续载荷（通常为 Stealer 或 RAT）
3. 建立持久化以维持对主机的控制
```

## 第8章：IOC 汇总

所有 IOC 必须集中在此章节，使用标准格式。

### IOC 列表格式

```markdown
## IOC 汇总

### 文件指标
| 类型 | 值 | 说明 |
|------|-----|------|
| MD5 | `eea30919ad0071a4eaffe4d227deb43c` | 样本哈希 |
| SHA256 | `66233ea314a3a5be48a3623f50de1566832fbd54d942b76c947533937fd02a81` | 样本哈希 |
| 文件名 | `sample_001.exe` | 原始文件名 |

### 网络指标
| 类型 | 值 | 说明 |
|------|-----|------|
| Domain | `evil[.]com` | C2 域名 |
| IP | `1.2.3[.]4` | C2 地址 |
| URL | `hxxp://evil[.]com/gate.php` | 回连地址 |

### 主机指标
| 类型 | 值 | 说明 |
|------|-----|------|
| Mutex | `Global\{GUID}` | 互斥体 |
| Registry | `HKCU\Software\Microsoft\Windows\CurrentVersion\Run\xxx` | 持久化键 |
| File Path | `%TEMP%\xxx.exe` | 释放路径 |

### 内嵌指标
| 类型 | 值 | 说明 |
|------|-----|------|
| MD5 (内嵌) | `49f9a15edb340f721ba66e6133ed05fb` | 样本中硬编码的哈希 |
| Base64 Config | `NZqRVuPR0FFU2sFc2oRy...` | 加密配置（未解密） |
```

## 第9章：关联分析

### 必须标明需要进一步分析的 IOC 及推荐的 skill

```markdown
## 关联分析

### 待深入分析的 IOC

| IOC | 类型 | 推荐 Skill | 分析目的 |
|-----|------|-----------|---------|
| `evil[.]com` | Domain | `/domain-analysis` | C2 域名威胁情报 |
| `1.2.3.4` | IP | `/ip-analysis` | C2 IP 归属和历史 |
| `hxxp://evil[.]com/gate.php` | URL | `/url-analysis` | 回连端点分析 |
| `NZqRVuPR0FFU2sFc2oRy...` | Base64 | 手动解密 | 提取加密配置 |

### 关联 Skill 调用建议

1. **C2 域名分析**:
   ```
   /domain-analysis evil.com
   ```
   - 检查域名注册信息
   - 查询威胁情报标签
   - 检测 DGA/同形字攻击

2. **C2 IP 分析**:
   ```
   /ip-analysis 1.2.3.4
   ```
   - 查询 IP 地理位置
   - 检查恶意标签
   - 关联历史恶意活动

3. **流量分析**（如有 PCAP）:
   ```
   /traffic-analysis capture.pcap
   ```
   - 提取实际 C2 通信
   - 分析协议特征
```

### 常见问题清单

```markdown
## 分析存在的问题

- [ ] C2 配置加密，未能提取明文地址

- [ ] 未在沙箱中执行，缺少运行时 IOC
- [ ] 编译时间戳可能被篡改，无法确定真实编译时间
- [ ] 缺少网络流量，无法验证 C2 通信协议
- [ ] YARA 规则可能存在误报
- [ ] 无法确认是否为红队工具或真实恶意软件
- [ ] 样本可能为多阶段攻击的一部分，缺少完整攻击链
```

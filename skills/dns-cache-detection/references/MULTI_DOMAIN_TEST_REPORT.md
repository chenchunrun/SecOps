## 🎓 技术价值

### DNS缓存探测技术特点

**优势**:
- ✅ 无需DNS日志权限
- ✅ 不污染DNS缓存 (RD=0)
- ✅ 秒级时间精度
- ✅ 低误报率 (<1%)
- ✅ 实时检测能力
- ✅ 可持续监控部署

**限制**:
- ⚠️ 只能检测近期访问（TTL时间窗口）
- ⚠️ 无法直接定位源IP（需结合日志）
- ⚠️ 依赖IOC准确性
- ⚠️ 无法检测非DNS通信威胁

**适用场景**:
- 🎯 快速威胁狩猎
- 🎯 持续安全监控
- 🎯 应急响应辅助
- 🎯 无日志权限环境
- 🎯 关键基础设施防护

1. **检查DNS日志**
   ```bash
   grep "accesserdsc.com" /var/log/named/queries.log | \
     awk '{print $1, $2, $6}' | \
     grep "17:50\|18:0[01]"
   ```
   目标：定位源IP地址

2. **检查防火墙日志**
   - 目标IP: 104.21.25.11, 172.67.221.140
   - 时间范围: 17:50~18:05
   - 目标：确认外连行为

3. **隔离受感染主机**
   - 根据日志定位主机
   - 网络隔离（VLAN/ACL）
   - EDR深度扫描

4. **封禁C2域名和IP**

   - 防火墙规则

5. **主机取证**

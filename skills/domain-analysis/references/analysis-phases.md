# 域名分析阶段详解

## 第一阶段：域名验证与预处理

### 格式验证
- 检查域名格式是否合法
- 处理 IDN（国际化域名）/ Punycode
- 提取有效 TLD

### IDN/Punycode 解码
```bash
python scripts/domain_validate.py xn--pple-43d.com
# 输出: аpple.com (西里尔字母 'а')
```

### 域名分类
| 类型 | 示例 | 说明 |
|------|------|------|
| 顶级域名 | example.com | 二级域名 |
| 子域名 | sub.example.com | 需要分析主域 |
| 动态 DNS | xxx.dyndns.org | 高风险 |
| 免费域名 | xxx.tk / xxx.ml | 高风险 |

## 第二阶段：DNS 分析

### DNS 记录查询
```bash
python scripts/domain_dns.py example.com --all
```

| 记录类型 | 用途 | 安全关注点 |
|---------|------|-----------|
| A | IPv4 地址 | 解析到的 IP 信誉 |
| AAAA | IPv6 地址 | 解析到的 IP 信誉 |
| MX | 邮件服务器 | 钓鱼邮件来源 |
| NS | 域名服务器 | DNS 托管商 |
| TXT | 文本记录 | SPF/DKIM/DMARC |
| CNAME | 别名 | 可能指向恶意域名 |

### 解析历史
- 历史解析 IP
- IP 变更频率
- Fast-flux 检测

## 第三阶段：WHOIS 分析

### 关键字段
| 字段 | 安全关注点 |
|------|-----------|
| 注册时间 | 新注册域名风险高 |
| 过期时间 | 即将过期可能被劫持 |
| 注册商 | 某些注册商被滥用 |
| 注册人 | 隐私保护 vs 真实信息 |
| 注册邮箱 | 可用于关联分析 |
| 域名服务器 | DNS 托管商 |

### 高风险特征
| 指标 | 高风险特征 |
|------|-----------|
| 注册时间 | < 30 天 |
| 隐私保护 | 启用 + 其他可疑特征 |
| 注册商 | 已知被滥用的注册商 |
| 注册邮箱 | 一次性邮箱 |

## 第四阶段：ICP 备案查询（中国域名）

调用 `cybersec-cloud` MCP 的 `intel_icp_lookup` 工具：
```
MCP 调用: intel_icp_lookup
参数: domain=<domain>
```

### 返回数据
| 字段 | 说明 |
|------|------|
| 备案号 | 如 京ICP备xxxxx号 |
| 主体名称 | 备案公司/个人 |
| 主体类型 | 企业/个人/政府 |
| 审核时间 | 备案审核通过时间 |

### 风险判断
- 无备案 + .cn 域名 = 可疑
- 备案主体与域名内容不符 = 可疑
- 个人备案 + 商业内容 = 需关注

## 第五阶段：威胁情报查询

调用 `cybersec-cloud` MCP 的 `risk_insight` 工具：
```
MCP 调用: risk_insight
参数: indicator=<domain>, kind=domain
```

### 返回数据解读
| 字段 | 说明 |
|------|------|
| 威胁标签 | phishing/malware/c2/spam 等 |
| 置信度 | 情报可信程度 |
| 首次发现 | 该域名首次被标记的时间 |
| 关联 IP | 解析到的 IP 列表 |
| 关联样本 | 与该域名通信的恶意样本 |

## 第六阶段：关联分析

| 发现的 IOC | 调用的技能 | 条件 |
|-----------|-----------|------|
| 解析 IP | `ip-analysis` | IP 数量 ≤ 5 时自动分析 |
| 关联 URL | `url-analysis` | 数量 ≤ 3 时自动分析 |
| 子域名 | 本技能 | 数量 ≤ 3 时自动分析 |

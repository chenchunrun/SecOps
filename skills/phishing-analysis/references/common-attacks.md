# 常见钓鱼攻击套路

本文档总结常见的钓鱼邮件攻击模式，帮助快速识别威胁。

## 套路一：密码保护压缩包

```
邮件：张总，这是报销单据，密码123456
附件：报销单.zip（加密）
里面：其实是.exe病毒
```

**为什么这样？** 因为杀毒软件扫不了加密的压缩包。

**识别要点**：
- 邮件正文提供解压密码
- 压缩包内含可执行文件
- 邮件内容极短，缺少正常商务邮件特征

---

## 套路二：伪装文件名

```
附件看起来是：合同.pdf
实际文件名：合同.pdf.exe
```

**为什么这样？** Windows默认隐藏扩展名，你以为打开的是PDF。

**识别要点**：
- 双扩展名文件
- 文件图标与扩展名不符
- 使用 Unicode 字符伪装（如使用 RLO 字符反转文件名）

---

## 套路三：冒充领导

```
发件人显示：王总 <ceo@company.com>
实际地址：wang_ceo@gmail.com
内容：紧急转账...
```

**为什么这样？** 看显示名不够，要看真实邮箱地址。

**识别要点**：
- 显示名和实际邮箱不匹配
- 使用免费邮箱冒充企业邮箱
- 内容涉及紧急转账、敏感操作

---

## 套路四：钓鱼链接

```
邮件：请点击链接验证您的账户
链接显示：https://www.bank.com/verify
实际跳转：https://www.bank-verify.evil.com/
```

**为什么这样？** 利用 HTML 让显示文本和实际链接不同。

**识别要点**：
- 鼠标悬停查看真实URL
- 检查域名是否为合法域名的子域
- 注意相似字符替换（如 0 替代 o）

---

## 套路五：二维码钓鱼 (Quishing)

```
邮件：扫码完成身份验证
附件：包含恶意链接的二维码图片
```

**为什么这样？** 二维码绕过传统URL扫描，移动设备防护较弱。

**识别要点**：
- 邮件中包含二维码图片
- 要求扫码进行敏感操作
- 无法预览二维码指向的URL

---

## 套路六：远程模板注入

```
附件：正常的.docx文档
打开时：自动从远程服务器下载恶意模板
```

**为什么这样？** 文档本身无恶意代码，绕过静态扫描。

**识别要点**：
- 打开文档时有网络请求
- 检查文档的外部链接关系

---

## 套路七：HTML附件

```
附件：invoice.html
打开后：显示伪造的登录页面，窃取凭据
```

**为什么这样？** HTML可以在本地运行JavaScript，模拟任何网站。

**识别要点**：
- .html/.htm 附件
- 打开后要求输入账号密码
- 页面模仿知名品牌

---

## 快速判断清单

| 特征 | 风险等级 |
|------|----------|
| 邮件正文提供解压密码 | 🔴 高危 |
| 附件是加密压缩包 | 🔴 高危 |
| 双扩展名文件 | 🔴 高危 |
| Reply-To 与 From 不同 | 🟠 可疑 |
| 包含二维码图片 | 🟠 可疑 |
| 要求紧急操作 | 🟠 可疑 |
| 使用短链接 | 🟡 注意 |
| 通用问候语 | 🟡 注意 |

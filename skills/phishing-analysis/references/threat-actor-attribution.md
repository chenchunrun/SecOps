# 钓鱼攻击组织归因指南

## 归因方法论

组织归因基于以下维度的特征匹配：

| 维度 | 特征类型 |
|------|----------|
| 投递渠道 | 邮件/IM/SEO/供应链 |
| 诱饵主题 | 财税/招聘/热点事件/官方通知 |
| 载荷类型 | 文档宏/LNK/CHM/MSI/EXE |
| 技术手段 | 白加黑/无文件/进程注入/DLL侧载 |
| 恶意软件 | RAT家族/C2特征 |
| 目标行业 | 金融/政府/医疗/制造 |
| 基础设施 | 域名模式/IP段/云服务 |

## 国内黑产组织

### 银狐 / 游蛇 / 谷堕大盗 (UTG-Q-1000)

**活跃时间**: 2022年9月至今

**目标**: 企业财务、税务、会计人员；金融、教育、电商行业

**归因指纹**:

| 特征 | 描述 | 置信度 |
|------|------|--------|
| 诱饵主题 | 财税相关：发票、税务稽查、补贴申领 | 高 |
| 文件名 | "金税"、"发票"、"稀企业名单"、"电子税务局" | 高 |
| 投递渠道 | 微信/QQ群、搜索引擎SEO、钓鱼邮件 | 高 |
| 载荷格式 | CHM、MSI、PyInstaller打包EXE | 中 |
| 技术特征 | 白加黑(NetSarang/腾讯游戏组件)、COM计划任务 | 高 |
| RAT家族 | Gh0st变种(Winos 4.0)、ValleyRAT | 高 |
| C2特征 | 大量使用香港IP、阿里云OSS | 中 |
| 后续行为 | 控制微信群继续传播、冒充身份诈骗 | 高 |

**典型攻击链**:
```
SEO投毒/微信群 → 财税主题诱饵 → CHM/MSI载荷
→ 白加黑加载 → 多阶段解码 → Gh0st/ValleyRAT
→ 键盘记录+屏幕监控 → 冒充身份诈骗
```

**检测规则**:
```
# 文件名关键词
金税|发票|税务|稀企业|电子税务局|补贴|退税

# 诱饵内容
偷漏涉税|违规企业名单|税务稽查|申报

# C2域名模式
*.oss-cn-*.aliyuncs.com (载荷托管)
```

## 境外APT组织

### Lazarus / APT38 / Hidden Cobra (朝鲜)

**活跃时间**: 2009年至今

**目标**: 金融机构、加密货币交易所、国防军工

**归因指纹**:

| 特征 | 描述 | 置信度 |
|------|------|--------|
| 诱饵主题 | 招聘/求职、加密货币投资、官方通知 | 高 |
| 社工手法 | 虚假招聘者LinkedIn/Telegram联系 | 高 |
| 文档漏洞 | CVE-2017-0199, CVE-2021-40444 | 中 |
| 投递方式 | 鱼叉邮件、社交媒体、水坑攻击 | 高 |
| RAT家族 | Dtrack, MATA, AppleJeus, TraderTraitor | 高 |
| 技术特征 | 多层混淆、多重跳转隐藏最终URL | 高 |
| 伪装对象 | Naver、Binance、招聘网站 | 中 |

**典型攻击链**:
```
LinkedIn虚假招聘 → 求职主题文档 → CVE漏洞利用
→ 多阶段加载器 → Dtrack/MATA RAT
→ 横向移动 → 金融窃取/加密货币盗窃
```

**检测规则**:
```
# 诱饵关键词
job offer|career opportunity|salary|interview
加密货币|投资机会|收益|钱包

# 域名特征
仿冒 Naver、Binance 等知名服务
使用 Azure Blob 托管钓鱼页面
```

### FIN7 / Carbanak (俄罗斯)

**活跃时间**: 2013年至今

**目标**: 零售、餐饮、金融；2024年扩展到汽车、媒体行业

**归因指纹**:

| 特征 | 描述 | 置信度 |
|------|------|--------|
| 诱饵主题 | IT工具下载、SEC文件、商业合作 | 高 |
| 攻击方式 | 仿冒软件下载站(Typosquatting) | 高 |
| 仿冒软件 | 7-zip, PuTTY, Notepad++, AnyDesk, AutoDesk | 高 |
| 投递渠道 | Google广告、鱼叉邮件+电话跟进 | 高 |
| RAT家族 | Carbanak, Gracewire, Anunak | 高 |
| 技术特征 | PowerTrash混淆shellcode、AvNeutralizer绕过EDR | 高 |
| 基础设施 | 4000+域名网络、shell域名转钓鱼站 | 高 |

**典型攻击链**:
```
Google广告/鱼叉邮件 → 仿冒软件下载页 → Dropbox托管载荷
→ 恶意安装包 → DLL侧载 + WAV文件shellcode
→ Carbanak/Gracewire → POS系统/金融数据窃取
```

**检测规则**:
```
# 仿冒域名模式
*advanced-ip-scanner*|*putty*|*notepad*
*7-zip*|*anydesk*|*autodesk*

# 载荷特征
PowerTrash shellcode invoker
AvNeutralizer EDR bypass
```

## 归因决策树

```
钓鱼邮件分析
    │
    ├─ 诱饵主题是财税相关？
    │   └─ 是 → 检查载荷格式
    │       ├─ CHM/MSI + 白加黑 → 银狐 (高置信)
    │       └─ 其他格式 → 需更多IOC
    │
    ├─ 诱饵主题是招聘/加密货币？
    │   └─ 是 → 检查社工渠道
    │       ├─ LinkedIn + 求职文档 → Lazarus (高置信)
    │       └─ 其他渠道 → 需更多IOC
    │
    ├─ 仿冒软件下载站？
    │   └─ 是 → 检查仿冒对象
    │       ├─ IT工具(PuTTY/7-zip等) → FIN7 (高置信)
    │       └─ 财税软件 → 银狐 (高置信)
    │
    ├─ 诱饵是政府/军事公文？
    │   └─ 是 → 检查目标地区
    │       ├─ 中国/巴基斯坦 → 响尾蛇/蔓灵花
    │       └─ 其他地区 → 需更多IOC
    │
    └─ 供应链投毒(NPM/PyPI)？
        └─ 是 → amdc6766 (中置信)
```

## 💡 技术洞察

### RD=0探测技术验证

**原理**:
```
RD=0 (Recursion Desired = 0)
├─ 非递归查询
├─ DNS只返回已缓存数据
├─ 缓存不存在 → 返回REFUSED
└─ 不触发上游查询 → 不污染缓存
```

**实验验证**:
- ✅ 连续10次RD=0查询，TTL正常衰减
- ✅ 证明探测行为未触发缓存刷新
- ✅ 证明TTL对比法的可靠性

### TTL对比法精度

**时间精度**: 秒级
**误报率**: < 1%
**适用场景**:
- ✅ 无DNS日志权限
- ✅ 快速威胁狩猎

